## 📝 Opis projektu

Niniejsze repozytorium zawiera główną, przyrostowo rozwijaną wersję projektu zaliczeniowego. Celem projektu jest stworzenie nowoczesnej, interaktywnej aplikacji webowej typu CV, prezentującej nabyte umiejętności z zakresu technologii frontendowych i backendowych.

Projekt ewoluuje od statycznego dokumentu HTML do dynamicznej aplikacji zintegrowanej z zewnętrznym serwerem.

---

## 🛠️ Stos technologiczny (Tech Stack)

- **Frontend:** HTML5, CSS3 (Custom Properties, Flexbox), JavaScript (ES6+)
- **Dane:** JSON, Fetch API
- **Trwałość danych:** Local Storage
- **Backend:** Formspree API (BaaS - Backend-as-a-Service)
- **Komunikacja:** Fetch API (metoda POST, JSON)
- **Narzędzia:** Git, GitHub Pages
